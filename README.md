# Aplicación Web Interactiva de Los Simpsons

Una aplicación web interactiva sobre Los Simpsons, construida con Next.js y React.

## Tecnologías Utilizadas

- **Framework**: [Next.js](https://nextjs.org/)
- **Biblioteca de UI**: [React](https://reactjs.org/)
- **Estilos**: [Tailwind CSS](https://tailwindcss.com/)
- **Componentes de UI**: [DaisyUI](https://daisyui.com/)
- **Animaciones**: [Framer Motion](https://www.framer.com/motion/)

## Puesta en Marcha

Para obtener una copia local y ponerla en funcionamiento, sigue estos sencillos pasos.

### Prerrequisitos

- Node.js (v18.17.0 o superior)
- npm (v9.0.0 o superior)

### Instalación

1.  Clona el repositorio
    ```sh
    git clone https://github.com/mariotc1/proyecto-ut6-the-simpsons.git
    ```
2.  Instala los paquetes NPM
    ```sh
    npm install
    ```
3.  Ejecuta el servidor de desarrollo
    ```sh
    npm run dev
    ```

La aplicación estará disponible en `http://localhost:3000`.

## Scripts Disponibles

- `npm run dev`: Inicia el servidor de desarrollo.
- `npm run build`: Compila la aplicación para producción.
- `npm run start`: Inicia un servidor de producción.
- `npm run lint`: Analiza el código con ESLint.
